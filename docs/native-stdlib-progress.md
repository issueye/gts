## 四、遇到的问题与解决

### 问题 1：类型系统差异

**问题描述**：最初代码使用了 `object.Integer` 和 `object.Float`，但 GTS 实际使用统一的 `object.Number` 类型。

**解决方案**：
- 将所有 `*object.Integer` 改为 `*object.Number`
- 将所有 `*object.Float` 改为 `*object.Number`
- 调整数值比较逻辑

### 问题 2：RegExp 结构字段差异

**问题描述**：代码使用 `re.Pattern` 但实际字段是 `re.Source`。

**解决方案**：
- 修改为使用正确的 `re.Source` 字段

### 问题 3：HashKey 方法缺失

**问题描述**：String 对象没有 `HashKey()` 方法。

**解决方案**：
- 使用 `object.HashKeyFor()` 辅助函数

### 问题 4：函数调用集成

**问题描述**：`callFunction()` 需要集成到 evaluator。

**当前状态**：
- 暂时返回 nil
- 标记为 TODO，待后续集成
